Byte Array to Image Converter

The input for the image height and width must be a natural number. For example 1, 2 or 300.

The input for the input matrix must be in the form '[X,Y,Z,...,X,Z,Y]' with X, Y, Z between 0 and 255

Attention: Only 3-channel RGB images are processed correctly. This means that a single pixel consists of 3 numbers, the red value, the green value and the blue value in exactly this order.

Example [1,2,3] is an image consisting of a pixel with the red value 1, the green value 2 and the blue value 3.
